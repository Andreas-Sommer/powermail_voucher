Users Manual
============

Target group: **Editors**

- How does it work?

  - Goto List view on a page of your choice

  - Create a campaign

    - Optional: add voucher codes manually

  - Goto BE Module Voucher Codes to action import

  - Select campaign and upload a .csv with your voucher codes

  - Create Powermail form and add a new field **Voucher Campaign** then switch to Campaign Tab and select one
